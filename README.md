# PostgreSQL & Power BI HR Analytics Dashboard

## Project Overview

This project demonstrates an end-to-end HR Analytics workflow using PostgreSQL for data preparation, validation, exploratory data analysis, and business analysis, followed by Power BI for developing an interactive HR Analytics dashboard.

The project analyzes employee attrition, workforce demographics, salary distribution, and department-level performance to generate meaningful business insights for HR decision-making.

---

# Objectives

- Import and manage HR data using PostgreSQL.
- Perform data preparation and validation.
- Conduct exploratory data analysis using SQL.
- Solve business problems using SQL queries.
- Build an interactive HR Analytics dashboard in Power BI.
- Generate actionable business insights.
